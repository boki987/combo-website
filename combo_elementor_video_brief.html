Važno za mobilne uređaje

Video autoplay je blokiran na iOS i Android browserima. U Elementoru isključi "Play on mobile" i postavi Fallback Image — screenshot iz videa koji izgleda isto. Na mobilnom se prikazuje slika, na desktopu video.

Alati za kompresiju videa (besplatno)
HandBrake (desktop) Clideo.com (online) FFmpeg (CLI)
Cilj: MP4, H.264, 1080p, ispod 5MB. U HandBrake: Web Optimized ON, RF 26–28.

Koraci implementacije

1
Preuzmi video sa Pexels
pexels.com → "server room blue" → Download 1080p MP4 (~50MB originalno)
2
Kompresuj sa HandBrake
H.264 · RF 27 · Web Optimized ON → cilj <5MB · snimi MP4
3
Napravi fallback sliku
Screenshot prvog frame-a videa → snimi kao screenshot.jpg
4
Upload u WordPress Media Library
Upload i MP4 i JPG → kopiraj URL oba fajla
5
Elementor → Section bg: Video
Paste MP4 URL · Loop ON · Mute ON · Play on Mobile OFF · Fallback = screenshot.jpg
6
Background Overlay + sadržaj
Overlay: #0d1a2e, 75% → dodaj Heading + Text + Button widgete iznad

Custom CSS — grid pattern efekat

Elementor → Section → Advanced → Custom CSS:
selector {
  background-image:
    linear-gradient(
      rgba(55,138,221,0.07) 1px,
      transparent 1px),
    linear-gradient(90deg,
      rgba(55,138,221,0.07) 1px,
      transparent 1px);
  background-size: 52px 52px;
}
